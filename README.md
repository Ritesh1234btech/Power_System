# Power_System

## Dashboard

The dashboard HTML is already stored in `index.html` at the repository root.

## How to open it

1. Open this repository in your editor.
2. Open `index.html`.
3. Preview it in a browser by opening the file directly or by using a live preview extension like Live Server.

## How to use it

1. Add generators using the coefficient inputs.
2. Enter system load demand `P_D`.
3. Click `Solve dispatch →`.
4. View the results and plots in the result panel.

## Notes

- The dashboard is pure HTML/CSS/JavaScript and runs entirely client-side.
- You can modify `index.html` to change the dashboard layout, styling, or solver behavior.

## Deployment

See `DEPLOY.md` for instructions to run locally and publish the dashboard online.
